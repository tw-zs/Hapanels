# Font Licenses

Fonts added for AOD clock styles:

- Monoton Regular: SIL Open Font License 1.1, from Google Fonts `ofl/monoton`.
- Bangers Regular: SIL Open Font License 1.1, from Google Fonts `ofl/bangers`.
- Big Shoulders Display variable weight: SIL Open Font License 1.1, from Google Fonts `ofl/bigshouldersdisplay`.
- Anton Regular: SIL Open Font License 1.1, from Google Fonts `ofl/anton`.

- Zaklad Regular: repository README says "SIL Open Font"; added from `warszawskie-kroje/zaklad` after project owner confirmation that Warszawskie Kroje are SIL licensed. Repository has no root `LICENSE` file as of 2026-07-07.
