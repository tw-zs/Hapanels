Szybka instalacja

Pobierz najnowszy APK z GitHub Releases i zainstaluj go na panelu albo tablecie.

Pobierz APK Repozytorium
  1. Pobierz APK z sekcji Releases.
  2. Zainstaluj aplikację na Shelly Wall Display albo tablecie Android.
  3. Połącz z Home Assistant i włącz MQTT, jeśli chcesz discovery panelu w HA.
  4. Na Shelly nadaj uprawnienie WRITE_SETTINGS, jeśli ma działać sterowanie jasnością.

O projekcie

Hapanels nie udaje Lovelace w WebView. Home Assistant dostarcza encje, konfigurację i usługi, a aplikacja renderuje panel natywnie na Androidzie.

Zasada sprzętowa: pokazujemy tylko realne, zweryfikowane funkcje urządzenia. Brak fikcyjnych sensorów temperatury, wilgotności albo odległości.

Strona statyczna deployowana z docs/ przez GitHub Actions. OAuth redirect: r1ha://auth-callback.
